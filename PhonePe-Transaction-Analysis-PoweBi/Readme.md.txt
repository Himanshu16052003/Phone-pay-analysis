# 📊 Phone Pay Transaction Analysis 

## 📌 Project Overview
This project presents an interactive Power BI dashboard for analyzing Phone pay transaction data. It provides insights into transaction performance, user behavior, service usage, and payment success rates.

## 🚀 Dashboard Features
- KPI Cards
  - Total Transactions
  - Total Transaction Value
  - Unique Users
  - Success Rate
- Monthly Transaction Trend
- Age Segment Contribution
- Service-wise Transaction Analysis
- Top 5 Users
- Weekday vs Weekend Usage
- Interactive Filters

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel

## 📈 Key Insights
- 300K total transactions
- ₹3.47 Billion transaction value
- 108K unique users
- 96% successful transactions
- Gen X and Millennials contributed the highest transaction volume
- Loan services generated the highest transaction value
